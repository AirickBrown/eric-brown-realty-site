# Project Overview

## Objective
Build a mobile-first real estate website focused on seller lead generation for Eric.

## Primary Business Goal
Convert homeowners into listing appointments by offering a home valuation / pre-listing package and creating fast follow-up workflows.

## Core Strategy
The website is not just informational. It is a lead capture and conversion system.

## Main Audience
- Homeowners thinking about selling
- Homeowners curious about their home value
- Secondary audience: buyers using IDX search

## Success Criteria
- Visitor submits home valuation form
- Contact information is captured
- CMA / pre-listing package is sent automatically
- Eric receives an immediate notification
- Lead is logged in Podio
- Eric follows up quickly by phone/text
