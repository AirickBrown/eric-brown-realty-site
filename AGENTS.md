# AGENTS.md

## Project Purpose
This project is a real estate website for Eric focused primarily on seller lead generation.

## Business Priorities
1. Capture seller leads
2. Send valuation / pre-listing materials automatically
3. Notify Eric immediately
4. Log leads in Podio
5. Maintain a strong trust-building brand presence
6. Provide IDX search as a secondary feature

## Required Features
- Home valuation form
- Automated email delivery
- Optional SMS workflow
- Podio integration
- Testimonials
- Mobile-first design
- Sticky call/text actions
- Seller-focused landing sections
- IDX search page

## Design Direction
- Clean
- Trustworthy
- Professional
- Mobile-first
- Conversion-oriented

## Implementation Guidance
- Prioritize working lead capture over visual extras
- Keep components modular
- Separate content from logic where possible
- Document integration points clearly
- Add TODO comments where external credentials or services are required
