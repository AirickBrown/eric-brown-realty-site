# Homepage Wireframe Spec

## Document Purpose
This document is the PDF-friendly wireframe and content specification for the homepage. It is intended to guide visual review before design or development begins.

## Page Summary
- Page name: Homepage
- Primary audience: Homeowners thinking about selling
- Secondary audience: Buyers
- Primary goal: Generate seller leads
- Primary CTA: `Get My Home Value`
- Secondary CTA: `Call or Text Eric`

## Core Positioning
Eric helps homeowners find the best way to sell based on their motivation, timeline, and pricing goals. He may help them list traditionally or explore a direct purchase path, depending on what best fits their situation.

## Success Criteria
- The page feels trustworthy within the first screen
- Sellers immediately understand that Eric is help-first, not pushy
- The valuation CTA is obvious and repeated
- The direct-contact path is easy on mobile
- Buyer content remains secondary

---

## Wireframe Overview

### Mobile-First Section Order
1. Hero
2. Valuation CTA
3. Why Work With Eric
4. Selling Paths
5. Offer Highlights
6. Testimonials
7. Simple Process
8. Direct Contact
9. Footer

### Desktop Structure
- Sticky top navigation
- Hero in two columns
- Main content in stacked full-width sections
- Repeated CTA placement every 2 to 3 sections

## Section 1: Hero

### Purpose
Establish trust immediately and make the seller-focused value proposition clear.

### Wireframe
```text
+--------------------------------------------------+
| LOGO                            MENU / NAV       |
+--------------------------------------------------+
| HEADLINE                                         |
| Get clarity on the best way to sell your home    |
|                                                  |
| SUPPORTING COPY                                  |
| Work with an agent who looks at your goals,      |
| timeline, and options before recommending        |
| a path forward.                                  |
|                                                  |
| [ Get My Home Value ]   [ Call or Text Eric ]    |
|                                                  |
| Trust line: listing help, direct-sale guidance,  |
| and practical next steps                         |
|                                                  |
| IMAGE: Eric portrait or polished local-home shot |
+--------------------------------------------------+
```

### Content Requirements
- One clear seller-focused headline
- One short paragraph of supporting copy
- Two CTA buttons
- One trust support line

### Layout Notes
- Mobile: stacked layout with CTA visible without extra scrolling
- Desktop: text left, image right
- Buttons should be large and easy to tap

- The CTA is unmistakable
- The tone feels calm, helpful, and professional

## Section 4: Selling Paths

### Purpose
Help sellers understand that more than one path may be available.

```text
+--------------------------------------------------+
| SECTION HEADING                                  |
| More than one way to move forward                |
|                                                  |
| SHORT INTRO COPY                                 |
|                                                  |
| [ Path Card ]            [ Path Card ]           |
| List Your Home           Explore a Direct Sale   |
|                                                  |

```

### Content Requirements
- One heading
- One short intro paragraph

- Keep both options clear and respectful

- Sellers can quickly understand the options-based approach
- The page still feels focused and not fragmented

## Mobile Behavior Requirements
- Sticky call and text actions should remain visible
- Main headline and CTA should appear in the first screen
- Section spacing should feel clean and easy to scan
- Buttons should be thumb-friendly
- No section should become too dense on small screens

## Visual Direction
- Clean
- Trustworthy
- Professional
- Conversion-oriented
- Mobile-first
- Warm and supportive, not flashy

## Content And Asset Checklist
- Eric headshot or approved photography
- Final testimonial copy
- Final contact information
- Brokerage and legal footer content
- Final language for offer highlights
- Decision on hero CTA format: short form or button only

## Open Decisions
- Should the hero use a short embedded form or only buttons?
- What exact wording should be used for the direct-sale option?
- Which testimonials should appear first?
- What brand colors and font direction should define the PDF mockup and later site design?

## Approval Summary
This homepage should feel like a seller-focused help page first and a real estate brand page second. The conversion path should be clear, the tone should be practical and supportive, and every major section should reinforce trust, clarity, and a clear next step.
