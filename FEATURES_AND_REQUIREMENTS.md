# Features and Requirements

## Core Features

### 1. Home Valuation Lead Form
Required fields:
- First Name
- Last Name
- Email
- Phone
- Property Address
- Optional notes

Purpose:
- Capture seller leads
- Trigger automated delivery of valuation / pre-listing materials
- Notify Eric immediately

### 2. Automated Delivery
After form submission:
- Send CMA / pre-listing package by email
- Send text message if SMS workflow is enabled
- Confirm submission to lead

### 3. CRM Integration
- Push lead into Podio
- Store source and timestamp
- Log contact details for follow-up

### 4. Instant Notification
- Notify Eric immediately after a lead submits
- Notification methods can include email, text, CRM task, or app alert

### 5. IDX Search
- Buyer-friendly property search
- Integrated into website without distracting from seller conversion focus

### 6. Trust Builders
- Testimonials
- Professional branding
- Market expertise messaging
- Seller education content

### 7. Mobile-First UX
- Sticky Call button
- Sticky Text button
- Fast load times
- Clear CTA sections
