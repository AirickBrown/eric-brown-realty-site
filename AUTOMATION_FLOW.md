# Automation Flow

## Seller Lead Workflow

1. Visitor lands on website
2. Visitor sees "What's My Home Worth?" call to action
3. Visitor submits valuation form
4. Website stores lead information
5. Automation triggers:
   - Send CMA / pre-listing package by email
   - Send SMS if enabled
   - Notify Eric immediately
   - Push lead into Podio
6. Eric follows up quickly by phone/text

## Priority
Speed to lead is critical.
The system should help Eric respond as fast as possible to increase listing appointment conversions.
